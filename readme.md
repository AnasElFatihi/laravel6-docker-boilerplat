##  Laravel6-Boiler-Plat

This is a basic configuration for Docker to use LARAVEL 6 with mysql as a Database.

N.B : here is some Tips in case you had some trouble running it :

<ul>
    <li> <strong> The app is running under port 9000   </strong></li>
    <li>Change the IP address of the environement variable named <strong>DB_HOST</strong> to your local IP </li>
    <li> Example : DB_HOST=192.168.1.5 </li>
</ul>
